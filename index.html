    Recent Projects

  • In 2023 Perry Thompson and I performed an in place security overhaul of a municipal network, rebuilding, subnetting, and setting up multiple vLANs.
    • We secured the different municipal departments and protected against unautorized data access.
    • We also setup new wireless SSIDs and further isolated against breaches while keeping the public segregated.
    • We did all of this with 99.9% uptime, only taking the network down long enough to reboot after hours.

  • In 2023 I moved an entire SQL database with more than 15,000 customer records from a Liquid Web Linux server to a Constant Contact database.
    • I did this with no downtime in marketing campaigns and no loss of customer records.
    • I also adjusted DNS records to migrate to a new website, new email, and new file servers with no losses or downtime.

    Networking

  • In 2022 Perry Thompson and I replaced ethernet connections, and built a new network for a large apartment complex in East Lansing, MI.
    • We built the whole network using UniFi technologies as per client request.
    • We built a new LAN, subnetted, and created vLAN's all within the UniFi Dream Machine Console.
    • Perry Thompson and I mirrored their pre-exsisting SSID's and brought the new network up inline with the old, resulting in 99.9% uptime and the network being down no longer than a re-boot.

  • In 2022 Ron Kramer, Perry Thompson, and I networked a Childcare Center in Central MI..
